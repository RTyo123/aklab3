# aklab3

## Options:
   
- **-V / --Value** with value after   
- **-L / --List** with list after   
- **-h / --help**   
- **-v / --version**   

## Example:
   
./lab3 --Value=10 -L1,2,3 --List=1,2 -hvlL10  
Value = 10  
List = 1,2,3  
Help.  
Program version is 1.0.  
./lab3: invalid option -- 'l'  
Unknown option.
